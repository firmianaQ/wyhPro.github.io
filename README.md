7要发表的文章一般以markdown的格式放在这里`_posts/`，你只要看看这篇模板里的文章你就立刻明白该如何设置。

文件头长这样:

```
---
layout:     post
title:      标题
subtitle:   "副标题"
date:       2019-12-12 12:00:00
author:     "作者"
header-img: "img/post-bg-2015.jpg"
header-mask:  0.3 遮罩层透明度
catalog:      true 开启目录
multilingual: true 开启多语言
tags:
    - 标签
---
